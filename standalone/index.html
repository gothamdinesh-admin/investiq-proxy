About the FIF rules
• The NZ$50,000 threshold is based on original cost of overseas investments (not current market value).
• NZ-listed, ASX-listed (some), and AU-unit-trust holdings are generally exempt.
• Two main calculation methods: FDR (Fair Dividend Rate, 5% flat) and CV (Comparative Value, cap at actual gain).
• InvestIQ shows the FDR estimate as the default conservative view. Your actual liability may be lower under CV.
• Always consult a qualified NZ tax adviser before filing.

ANZ & BNZ — via Akahu Open Banking

          
Connects live to NZ banks. Free tier available.

        

        Recommended
      

      

        Akahu is NZ's open banking API — connects to ANZ, BNZ, ASB, Kiwibank, Westpac, and Sharesies.
        You authenticate directly with your bank (Akahu never sees your password).
      

      

        
Setup (5 minutes):

        

          1. Register a free app at akahu.nz/developers

          2. Get your App Token and User Token

          3. Enter them below — your holdings sync automatically
        

      

      

        

          
          
        

        

          
          
        

      

      
      
      

        
⚡ One-time setup required:

        

          1. Open a terminal / Command Prompt in Personal Investments\webapp

          2. Run: npm install

          3. Copy .env.local.example → .env.local and add your Claude API key

          4. Run: npm run dev

          5. Come back here and click Sync again ✓
